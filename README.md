# car_price_assigment#
🚗 Car Price Analysis using Python & Pandas

## 📌 Project Overview

This project is a simple **Car Price Analysis** assignment created using **Python** and the **Pandas** library. It demonstrates how to load, analyze, and extract useful information from a car dataset containing details such as car names, models, prices, and other attributes.

The project is implemented in **Google Colab**, making it easy to run without any local setup.

---

## 📂 Dataset

The dataset contains information about cars, including:

- Car Name
- Model
- Price
- Brand
- Year (if available)
- Other vehicle details

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- Google Colab

---

## 📊 Features

The notebook performs several data analysis tasks, such as:

- Loading the dataset using Pandas
- Displaying dataset information
- Finding specific car models
- Searching car prices
- Filtering cars based on different conditions
- Viewing selected columns
- Basic data exploration
- Data cleaning (if required)
- Using Pandas functions for analysis
